## SEO & Performance

The site is fully optimized for search engines and social sharing:

- ✅ **Structured Data** — JSON-LD schemas on every page (`ProfessionalService`, `WebSite`, `FAQPage`, `ItemList`, `Person`, `ContactPage`)
- ✅ **Open Graph & Twitter Cards** — Rich social media previews with custom OG image
- ✅ **Canonical URLs** — Prevents duplicate content indexing
- ✅ **Sitemap & Robots.txt** — Proper crawl directives with priority-based sitemap
- ✅ **Semantic HTML** — `<nav>`, `<main>`, `<section>`, `<footer>` with ARIA labels
- ✅ **Responsive Design** — Mobile-first, works across all screen sizes
- ✅ **Fast Loading** — No frameworks, no bundle — just optimized vanilla code

- 🎨 **Warm, editorial aesthetic** — Off-white backgrounds with dark typography
